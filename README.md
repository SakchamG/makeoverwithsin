# Makeover with Sin
Premium Makeup & Beauty Services

This is a Vite + React application built for the Makeover with Sin platform. The site offers a responsive, feature-rich interface to explore beauty services, view galleries, and book appointments.

## Designed for Mobile

The UI is built with a **mobile-first** approach using Tailwind CSS. It is fully responsive and guarantees a smooth experience on any device, from laptops down to thin mobile screens. The global container configuration ensures proper spacing automatically across breakpoints.

## Features

- **Responsive Components:** Fluid typography, grid elements, and padding via Tailwind CSS utility classes.
- **Dark Mode Support:** Configured via next-themes and integrated with Tailwind CSS contexts.
- **Fast Performance:** Bundled using Vite to offer blazing-fast HMR and optimized production code.

## Getting Started

To get started locally with the project, simply follow these commands:

### 1. Install Dependencies
If you have just cloned the project or pulled recent changes, install the dependencies first using npm:

```bash
npm install
```

### 2. Run the Development Server
Once packages are installed, spin up the local development server:

```bash
npm run dev
```

The application will usually run on `http://localhost:5173/`. Open this in your browser to view the app!

### 3. Build for Production
To build your app for production deployment:

```bash
npm run build
```
